### 一、安装vue3脚手架

- **卸载vue2脚手架**

  ```
  npm uninstall -g vue-cli
  ```

- **清除缓存**

  ```
  npm cache clen --force
  ```

- **安装最新脚手架**

  ```
  npm install -g @vue/cli
  ```

- **查看脚手架版本**

  ```
  vue -V
  ```
